عند إضافة مادة جديدة، سيتم إنشاء 8 فصول تلقائياً:

- الفصل الخامس

- الفصل السابع

### 2️⃣ **للمواد الموجودة (يدوياً)**

## 🚀 كيفية الاستخدام:

### للمواد الجديدة:

1. **حدّث صفحة الأدمن** (Ctrl+F5)
2. اذهب لـ "المواد"
3. أضف مادة جديدة (مثلاً: "الكيمياء")
4. **تلقائياً** سيكون للمادة 8 فصول!

**جرب الآن:**
- افتح صفحة الأدمن
- أضف مادة جديدة
- اذهب لـ "إدارة المادة"
- **ستجد 8 فصول جاهزة! ✨**

## 🎯 الخطوات التالية:

### 1. اختبر المواد الجديدة:
```
1. افتح صفحة الأدمن
2. اذهب لـ "المواد"
3. أضف مادة جديدة: "الكيمياء - الصف الأول"
4. اضغط "إدارة المحتوى"
5. ستجد 8 فصول جاهزة! ✅
```

### 2. أضف فصول للمواد القديمة:
```
1. افتح: add_8_chapters.html
2. اختر "الفيزياء" (أو أي مادة قديمة)
3. اضغط "إضافة 8 فصول"
4. تم! ✅
```

### 3. أضف محتوى للفصول:
```
1. اذهب لـ "المحتوى" في الأدمن
2. اختر المادة
3. اختر الفصل (مثلاً: "الفصل الأول")
4. أضف محاضرات أو امتحانات
5. كرر لباقي الفصول
```
